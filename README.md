# Cursor Cafe

A minimal VS Code extension scaffold built with TypeScript, esbuild, and Vitest.

## Features

- Contributes a custom activity bar container: `cursorCafe`
- Contributes a sidebar view: `cursorCafe.sidebar`
- Registers a refresh command: `cursorCafe.refresh`

## Development

```bash
npm install
npm run compile
```

To run in watch mode:

```bash
npm run watch
```

To execute tests:

```bash
npm test
```
